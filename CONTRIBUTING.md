# Contributing to bloomreach-buddy

Thanks for your interest in contributing!

## Getting Started

1. Fork the repo
2. Create a feature branch (`git checkout -b feat/my-feature`)
3. Commit your changes (commits must be signed)
4. Push to your fork and open a Pull Request

## Requirements

- All commits must be signed (SSH or GPG)
- All PRs must pass CI (lint, typecheck, tests)
- PRs are squash-merged into main

## Code Style

- TypeScript with strict mode
- ESLint for linting
- Prettier for formatting (if configured)

## Reporting Issues

Use GitHub Issues. Bug reports and feature requests welcome.

## AI-Driven Development

This project uses autonomous AI agents for development. Issues labeled `ai-task` are picked up by the orchestrator. If you see `ai-in-progress`, an agent is working on it.

You're welcome to contribute alongside the AI agents. Just make sure your PR doesn't conflict with in-progress AI branches (`ai/*`).
